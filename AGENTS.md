# Repository Agent Guide

This repository has two active work areas:

- `despoiler-spacetime/`: Rust + SpacetimeDB backend.
- `despoiler-pixijs/`: PixiJS client/application area.

## Scope and priorities

- Prefer making changes inside only the area requested by the task.
- Follow nested `AGENTS.md` files for area-specific rules.
- Keep changes focused; do not refactor unrelated files.
